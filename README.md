# GuadalIA M2
## Descripción:
Una IA escrita en Python 3 con un nombre inspirado en el río Guadalquivir.
## ¿Por qué hemos sacado el M2 (Model 2)?
Porque la anterior versión, el M1 (Model 1), se ha vuelto imposible de añadir mejoras porque he parado de entenderlo.
## ¿Qué diferencias tiene respecto a GuadalIA M1?
Tiene una forma diferente de procesar las frases con más eficiencia, manejandolas como listas en vez de como texto. Además, ahora ha cambiado el entrenamiento para tener mejores respuestas. Aunque desarrolla menos sus respuestas, ahora tienen más sentido.
## ¿Cómo lo uso?:
Abre un terminal dentro de la carpeta del proyecto y ejecuta:
```
python3 main.py
```
Cuando hagas esto, el modelo se empezará a entrenar y cuando acabe podrás empezar a usarlo.
